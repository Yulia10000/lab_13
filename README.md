# Лабораторна робота: Архітектура ПЗ:моноліт, мікросервіси та патерни
## Варіант 4: Сервіс доставки їжі
###Крок 1. Аналіз бізнес-домену
Сервіс доставки їжі — це система, яка дозволяє користувачам:
переглядати ресторани та меню;
формувати замовлення;
оплачувати онлайн;
відстежувати статус доставки;
отримувати сповіщення.

Основні учасники системи:
Користувач (клієнт)
Ресторан
Кур’єр
Адміністратор

Система повинна забезпечувати:
зручний інтерфейс
швидку обробку замовлень
безпечні платежі
координацію між рестораном і кур’єром
